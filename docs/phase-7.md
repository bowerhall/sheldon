# Phase 7: Mobile

**Timeline: 4-6 weeks**
**Depends on: Phase 5**
**Goal: Native iOS + Android apps with generative UI**

## Tasks

### 1. Mobile API (Week 1)
- REST/WebSocket endpoint, token-based auth
- Push notification infrastructure (FCM + APNs)

### 2. iOS App (Weeks 2-3)
- Swift/SwiftUI, chat + voice + push
- Generative UI: dynamic cards for skill results

### 3. Android App (Weeks 3-4)
- Kotlin/Jetpack Compose, same feature set

### 4. Generative UI (Weeks 4-5)
- Sheldon returns structured JSON alongside text
- Apps render as interactive cards, charts, forms

### 5. Polish (Week 5-6)
- Offline queuing, local caching, crash testing

## Success Criteria
- [ ] iOS and Android apps with chat + voice
- [ ] Push notifications for proactive messages
- [ ] Generative UI renders skill output
