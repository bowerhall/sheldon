# Interfaces

## Phase 0: Telegram

- Primary interface for text + voice
- Inline keyboards for structured interactions
- Natural language interface (no slash commands)

## Phase 5: Mac Menu Bar App

- SwiftUI app, keyboard shortcut activation
- WebSocket connection to Sheldon
- Push-to-talk voice, chat history
- Animated status indicator

## Phase 7: Mobile Apps

- iOS (Swift/SwiftUI) + Android (Kotlin/Jetpack Compose)
- Push notifications for proactive messages
- Generative UI: dynamic cards from structured JSON
- Offline message queuing

## Future: Custom Generative UI

- React-based web interface
- Sheldon returns structured data alongside text
- Client renders as interactive components
- Dashboard, charts, forms, card layouts
